--- Installation and Requirements ---
You will need to have a full version of Heroes of Might and Magic IV game to be able
to play.

You need to manually copy the subdirectories 'DATA' and 'MAPS' from the original game
directory to the engine4heroes installation directory.

If you are running engine4heroes on Windows and see complaints about missing DLLs,
then you may need to install the Microsoft Visual C++ redistributable package.
You can download it using the following URLs:

https://aka.ms/vs/17/release/vc_redist.x86.exe - for 32-bit x86 engine4heroes builds;
https://aka.ms/vs/17/release/vc_redist.x64.exe - for 64-bit x64 engine4heroes builds.

Or you can install the game using the installer, in this case, this package will be
installed automatically.
