# macOS

This directory contains scripts and configuration files for managing engine4heroes on macOS:

* `Brewfile`: Defines the basic development dependencies required to build engine4heroes, including SDL2 and related libraries
* `Brewfile.app_bundle`: Additional dependencies needed for creating macOS application bundles
* `cleanup_app_bundle_files.sh`: A utility script to clean up engine4heroes application data from the user's home directory

These files are used during development and deployment of engine4heroes on macOS systems.
